# The Battleship Game

A two-player, turn-based Battleship game played directly in the console.

## How to Play

1. Player Blue places ships on the board first, one by one.
2. When Player Blue finishes placement, press Enter to clear the screen.
3. Player Red places ships on the board, one by one.
4. When Player Red finishes placement, press Enter to clear the screen.
5. The game then continues in turns. Each player clears the screen before passing the turn to avoid revealing their board.
6. All other rules follow standard Battleship rules.

## Important Notes

- Make sure to press Enter when prompted to hide your board from the other player.
- Play proceeds turn by turn with hidden boards.

## Tech Stack

- Language: C++[cite: 4]
- Interface: Command Line Interface (CLI)

## Files

- `Battleship.cpp`: The source code containing game logic
- `Battleship.exe`: Pre-compiled file to run the game directly on Windows.
